## Getting Started

### Clone the repository
```bash
$ git clone https://github.com/yemiwebby/nest-bot-pusher.git
```

### Change directory
```bash
$ cd nest-bot-pusher
```

### Install dependencies
```bash
$ npm install
```

## Prerequisites
A basic knowledge of TypeScript and Node.js
